# UI Usage Guide

## What users can do
- enter a question
- choose retrieval depth
- choose context depth
- optionally add metadata filters
- view answer
- view latency
- view citations

## Available filters
These match the fields used in your Week 3 notebook:
- department
- document_type
- category
- region
- year
- source
- file_name

## Recommended workflow
1. Ask the question without filters first.
2. If the answer is broad, apply metadata filters.
3. Reduce or increase:
   - top_k_retrieval
   - top_k_context
4. Verify the citations shown in the UI.

## Example query
What are the onboarding compliance requirements for HR policy documents?
